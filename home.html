About Me

Hi! My name is Jeff and I am a student at UC Berkeley pursuing a degree in Computer Science and Statistics. I love messing around with web and mobile design and hacking things to see what happens. I am currently developing a dashboard to collect information about Indian citizen-government interaction and an Android application to improve the efficiency of private gardens.

I produce electronic music and am interested in creating software to auto-mix dance music. Check out my Soundcloud linked on the left! I've also grown a new fascination with cosmology after binge watching The Cosmos; part of my website is dedicated to some truly awesome pictures of our universe.

This site was built using a template from Wrapbootrap. If you like this website and want to build something similar, I definitely would recommend browsing their selection of Bootstrap templates.

Festiv

Festiv is a web application I developed that uses SoundCloud’s API to deliver EDM music to users without the hassle of searching through albums or playlists by using curated playlists and learning user preferences. I deployed the site using Django and mySQL.

  • Demo: http://festiv.me
  • Repo: http://github.com/jeffzheng1/Festiv

Beautiful Search Trees

Web application I developed to help students learn the insertion, deletion, and search methods of popular tree data structures through animation done in D3.

  • Demo: http://beautifulsearchtrees.herokuapp.com
  • Repo: http://github.com/jeffzheng1/BeautifulSearchTrees

Bribecaster

The dashboard in development for my CS research project. I created back-end models for storing citizen data and developed a dashboard interface using various data visualization packages. I created a communications network using Twilio to send robocalls and SMS messages to collect citizen information.

  • Repo: http://github.com/jeffzheng1/Bribecaster

Algovision & the 164 Language

Web application that I built that visualizes the call stack of algorithms written in 164, an object oriented computer language my team created throughout the Fall 2014 semester for my programming language class. It uses the Wolfram API to analyze the approximate runtime of a function from the structure of the call stack and plots the runtime function on a graph.

  • Repo: http://github.com/jeffzheng1/algovision
  • Video Demo: http://www.youtube.com/

SeedIt

An Android gardening application in development for the Citris Mobile App Challenge. Users receive customized recommendations based on location, time, and weather to optimize their harvest. I am leading the production of the application and am responsible for developing the backend models for the app’s plant database and designing the app’s user interface.

  • Link: http://getseedit.com

California Records

Website that I designed for an independent record label with two other Berkeley classmates. I managed bookings and production sessions. Our Youtube channel (https://www.youtube.com/c/CaliforniaRecords)has recorded 250,000 views and 1,700 subscribers.
