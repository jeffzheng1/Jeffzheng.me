About Me

Festiv

Festiv is a website I developed that uses SoundCloud’s API to deliver EDM music to users without the hassle of searching through albums or playlists by using curated playlists and learning user preferences. Deployed using Django and mySQL.

  • Demo: http://festiv.herokuapp.com
  • Repo: https://github.com/jeffzheng1/Festiv

Beautiful Search Trees

Web application I developed to help students learn the insertion, deletion, and search methods of popular tree data structures through animation done in D3.

  • Demo: https://beautifulsearchtrees.herokuapp.com
  • Repo: https://github.com/jeffzheng1/BeautifulSearchTrees

Bribecaster

Research project done in conjunction with Berkeley’s CS and Public Policy Department to analyze corruption in India and create actionable data for government usage. I created back-end models for storing citizen data and developed the dashboard interface using various data visualization packages.

  • Repo: https://github.com/maxwolffe/Bribecaster

AlgoVision & the 164 Language

Web application that I built that visualizes the call stack of algorithms written in 164, an object oriented computer language my team created throughout the Fall 2014 semester for my programming language class. It uses the Wolfram API to analyze the approximate runtime of a function from the structure of the call stack and plots the runtime function on a graph.

  • Repo: https://github.com/jeffzheng1/algovision

SeedIt

A garden planning Android application being developed for the Citris Mobile App Challenge. Users receive custom data based on location, time, and weather to provide optimal data for a successful harvest. I am designing the human user interface for the product.
